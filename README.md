## Descripción

<p align="justify">
    La presente aplicación busca obtener la ubicación del usuario y compartirla en tiempo real.
</p>

<p align="justify">
    Para lograr esto hacemos uso de una DB No Relacional que nos provee <i>Firebase</i>.
</p>
